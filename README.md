# NIS2 Srbija — Alat za samoprocenu usklađenosti

Interaktivni alat za samoprocenu obaveza prema Zakonu o informacionoj bezbednosti Srbije (ZIB 2025), usklađenom sa NIS2 direktivom.

## Kako koristiti

1. Otvorite [nis2.rs](https://nis2.rs) u pretraživaču
2. Odgovorite na nekoliko pitanja o vašoj organizaciji
3. Alat će odrediti:
   - Da li podležete zakonu
   - Vašu kategoriju (Priority / Important)
   - Obaveze, rokove i zahteve
   - Moguće kazne
